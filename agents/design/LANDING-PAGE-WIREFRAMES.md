## Notes for Developers

1. **Spacing System:**
   - Use rem-based spacing for consistency
   - All spacing follows 8px grid (0.5rem, 1rem, 1.5rem, etc.)

2. **Responsive Strategy:**
   - Mobile-first CSS
   - Use flexbox for layout flexibility
   - CSS Grid for card grids

3. **Interactive Elements:**
   - All hover effects use `transition: all 0.3s ease`
   - Lift effect: `translateY(-2px)` for buttons, `translateY(-4px)` for cards
   - Focus states: 2px outline in teal color

4. **Accessibility:**
   - All accordions keyboard-accessible
   - Modal trap focus when open
   - Skip-to-content link at top
   - All images have alt text

5. **Performance:**
   - Lazy load images below fold
   - Inline critical CSS
   - Defer non-critical JavaScript
   - Use WebP with JPEG fallback
