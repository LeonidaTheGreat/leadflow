### 2. TypeScript Error with PromiseLike (BLOCKING)
**Location:** `product/lead-response/dashboard/app/api/auth/trial-signup/route.ts` lines 178-210

**Problem:** Code used:
```typescript
void Promise.resolve(supabase.from('events').insert(...)).catch(...)
```

Supabase's `.insert()` returns a `PromiseLike`, not a full `Promise`. `Promise.resolve()` preserves the `PromiseLike` type which doesn't have `.catch()`.

**Fix:** Changed to async IIFE pattern:
```typescript
void (async () => {
  try {
    await supabase.from('events').insert(...)
  } catch (err) {
    console.error(...)
  }
})()
```
