## Color Palette

### Dark Mode (Primary)
```
Background:           slate-950     (#020617)
Card Background:      slate-800/50  (#1e293b80)
Card Border:          slate-700/50  (#33415580)
Text Primary:         white         (#ffffff)
Text Secondary:       slate-300     (#cbd5e1)
Text Muted:           slate-400     (#94a3b8)
Text Disabled:        slate-600     (#475569)

Primary (CTA):        emerald-500   (#10b981)
Primary Hover:        emerald-600   (#059669)
Primary Light:        emerald-400   (#34d399)

Secondary:            blue-500      (#3b82f6)
Accent:               purple-500    (#a855f7)

Success/Check:        emerald-400   (#34d399)
```

### Card States

**Default Card:**
```
background: slate-800/50
border: 1px solid slate-700/50
hover: border-slate-600, scale(1.02)
transition: all 200ms
```

**Highlighted Card (Pro):**
```
background: gradient-to-br from-slate-800 to-slate-900
border: 2px solid emerald-500/50
ring: 2px solid emerald-500/20
overlay: gradient-to-r from-emerald-500/5 to-blue-500/5
scale: 1.05
```

### Badges

```
Most Popular:  bg-emerald-500, text-white
Team Favorite: bg-blue-500, text-white
Enterprise:    bg-purple-500, text-white
```

## Spacing Scale

```
Section padding:     py-16 (64px)
Container max-width: max-w-7xl (1280px)
Card padding:        p-8 (32px)
Card gap:            gap-6 (24px)
Feature list gap:    space-y-3 (12px)
Section gap:         mb-20 (80px)
```

## Typography Scale

```
Section Title:       text-4xl md:text-5xl font-bold
Section Subtitle:    text-xl text-slate-400
Card Title:          text-2xl font-bold
Card Description:    text-sm text-slate-400
Price:               text-5xl font-bold
Price Unit:          text-slate-400
Feature Text:        text-sm text-slate-300
Category Header:     text-xs font-semibold uppercase tracking-wider
```

## Responsive Breakpoints

```
Desktop (1280px+):   4-column pricing grid
Tablet (768-1279px): 2x2 pricing grid
Mobile (<768px):     Single column, horizontal scroll for table
```

## Animation Specs

```
Card Hover:
  - transform: scale(1.02)
  - transition: all 200ms ease-out
  - shadow increase on hover

Button Hover:
  - transform: scale(1.05)
  - shadow: shadow-lg
  - transition: all 200ms

Highlighted Card:
  - Subtle pulse animation on ring (optional)
  - Scale: 1.05 (always elevated)

Table Row Hover:
  - background: slate-700/30
  - transition: colors 150ms
```

## Iconography

```
Tier Icons:
  Starter:   Zap (lightning bolt)
  Pro:       Star
  Team:      Users
  Brokerage: Building2

Feature Icons:
  Included:  Check (emerald-400)
  Excluded:  Minus (slate-600)
  CTA:       ArrowRight
```

## Interactions

### Billing Toggle
```
State: monthly | annual
Active: bg-emerald-500, text-white
Inactive: text-slate-300, hover:text-white
Transition: all 200ms
```

### FAQ Accordion
```
Closed: show question only
Open: show question + answer
Toggle: +/- icon rotation
Transition: height 200ms ease-out
```

### CTA Buttons
```
Primary (Highlighted):
  - bg-emerald-500
  - hover:bg-emerald-600
  - hover:shadow-lg hover:shadow-emerald-500/25
  - Full width in cards

Secondary:
  - bg-slate-700
  - border border-slate-600
  - hover:bg-slate-600
```
