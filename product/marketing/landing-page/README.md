# LeadFlow AI Landing Page

## Overview
A high-converting landing page designed for real estate agents interested in AI-powered lead management.

## Files
- `index.html` - Main landing page (24KB)
- `styles.css` - Complete styling with responsive design (16KB)
- `script.js` - Form handling, FAQ accordion, animations (5KB)

## Sections
1. **Hero** - Value proposition with 60s response time focus
2. **Stats Bar** - Key metrics (60s response, 2x conversion, etc.)
3. **Problem** - Pain points real estate agents face
4. **Features** - 4 AI agents (Response, Qualification, Nurture, Booking)
5. **Demo** - Interactive chat preview showing AI conversation
6. **Testimonials** - Social proof from early users
7. **Pricing** - 3 tiers (Starter $297, Growth $597, Team $1,297)
8. **FAQ** - Expandable accordion with common questions
9. **Signup** - Lead capture form with validation

## Key Features
- ✅ Mobile-responsive design
- ✅ Fast loading (<2s target)
- ✅ Form validation & formatting
- ✅ FAQ accordion
- ✅ Smooth scroll navigation
- ✅ Intersection observer animations
- ✅ LocalStorage form backup

## Form Fields
- Full Name
- Email
- Phone (auto-formatted)
- Brokerage
- Monthly lead volume
