## Design Guidelines

### Color Palette
- Primary: #6366f1 (Indigo)
- Background: #ffffff / #f9fafb
- Text: #111827 / #6b7280
- Dark Section: Gradient #1f2937 → #111827
- Accent: #f59e0b (Yellow for badge)

### Typography
- H1: 52px desktop / 36px mobile, weight 800
- H2: 36px desktop / 28px mobile, weight 700
- Body: 18px desktop / 16px mobile

### Responsive
- Mobile: Single column, stacked CTAs
- Tablet: 2-column where appropriate
- Desktop: Full layouts
