## UTM Tracking

Use these parameters to track traffic:

```
?utm_source=[reddit|facebook|linkedin]&utm_medium=social&utm_campaign=[campaign_name]&utm_content=[post_type]
```

Examples:
- `?utm_source=reddit&utm_medium=social&utm_campaign=data_post`
- `?utm_source=facebook&utm_medium=social&utm_campaign=agent_round_table`
- `?utm_source=linkedin&utm_medium=social&utm_campaign=personal_brand`
